🎯 Objetivo de Aprendizaje

Los estudiantes podrán: comprender la transferencia de formatos entre diferentes herramientas de IA, y usar ChatGPT, Gemini o Claude para diseñar prompts híbridos que combinen Role Prompting y Few-Shot con el fin de crear guiones en Markdown.

🤖 Prompt de Referencia para XOLO (Solo para Maestro)

⚠️ NOTA: XOLO ya conoce el tema de esta misión. Este prompt es SOLO de referencia en caso de emergencia técnica: 
Explícame por qué combinar Role Prompting + Few-Shot vuelve un prompt más robusto (más control, menos ambigüedad). Analicemos cómo definir un rol con agenda clara y cómo elegir ejemplos que "fijen" el estilo/estructura. Guíame para adaptar el mismo prompt a dos "idiomas" distintos: (1) un LLM para guion/Markdown y (2) una herramienta de presentaciones que interpreta contenido de otra forma.

📊 Diapositiva 3: Brain Spark

1. 💬 Pide a XOLO que te ayude a crear:

"Ayúdame a construir mi prompt combinado (Rol + Few-Shot) para un objetivo útil (presentación, proyecto o estructura web). Guíame a producir un guion en Markdown y a revisar si está listo para pegarse en Gamma/Tome (secciones claras, bullets, mensajes clave). Ejecuta: entrega mi prompt final + el guion/plan en Markdown en su versión lista para usar."

2. 🔬 Prueba tu prompt en ChatGPT, Gemini o Claude y reflexiona:

  • ¿Funciona el prompt como esperabas al generar el guion?
  • ¿Qué pasa si cambias el rol de la IA a uno completamente opuesto?
  • ¿Funciona igual el prompt en ChatGPT que en Gemini o Claude?
  • ¿Cómo facilita el uso de Markdown la transferencia de formatos entre diferentes herramientas de IA?
  • ¿De qué manera el uso de Role Prompting te dio un mayor control sobre la intención de la IA en comparación con un prompt simple?

¡Es hora de crear algo increíble! Ahora que entienden la teoría, vamos a la práctica. XOLO los ayudará a diseñar un prompt híbrido que combine un rol específico con ejemplos claros. Por ejemplo, podrían diseñar un prompt para crear una presentación sobre sostenibilidad que use un tono inspirador y datos precisos. Una vez que tengan su prompt, pruébenlo en ChatGPT, Gemini o Claude para generar un guion en Markdown. Verifiquen si está listo para usarse en herramientas como Gamma o Tome. Recuerden usar las preguntas de experimentación para ver cómo pequeños cambios en el rol o los ejemplos transforman el resultado. Si tienen dudas, pregúntenle a XOLO, él los ayudará a refinar su estrategia.
